# Selectable Table React App

A React application featuring a selectable table with a "Select All" checkbox functionality.

## Features

- ✅ Selectable table rows with individual checkboxes
- ✅ "Select All" header checkbox that selects/deselects all rows
- ✅ Indeterminate state for header checkbox when some rows are selected
- ✅ Visual feedback with highlighted selected rows
- ✅ Modern, responsive design

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to the URL shown in the terminal (usually `http://localhost:5173`)

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
